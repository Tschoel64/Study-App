# Quadratische Gleichungen

## Allgemeine Form
$$ax^2 + bx + c = 0$$

## abc-Formel
$$x = (-b \pm \sqrt{b^2-4ac}) / 2a$$

## Diskriminante
$$D = b^2 - 4ac$$

| D | Lösungen |
|---|----------|
| D > 0 | 2 Lösungen |
| D = 0 | 1 Lösung |
| D < 0 | keine |

## Beispiel
> x² - 5x + 6 = 0 → x₁ = 3, x₂ = 2
